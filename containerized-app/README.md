Judul
Containerized Application Menggunakan Docker dan Podman

Deskripsi
Task ini membahas proses pembuatan aplikasi Python sederhana yang dijalankan dalam container menggunakan Docker dan Podman. Containerization bertujuan untuk memastikan aplikasi dapat dijalankan secara konsisten di berbagai lingkungan.

Tujuan

Memahami konsep containerization

Membuat image aplikasi menggunakan Docker

Menjalankan aplikasi dalam container

Membandingkan penggunaan Docker dan Podman

Tools dan Teknologi

Python

Docker

Podman

Flask

Langkah Implementasi

Membuat aplikasi Python sederhana

Membuat Dockerfile untuk build image

Menjalankan container menggunakan Docker

Menjalankan container menggunakan Podman

Hasil Pengujian
Aplikasi berhasil dijalankan di dalam container dan dapat diakses melalui browser atau terminal. Proses build dan run berjalan dengan baik baik menggunakan Docker maupun Podman.
